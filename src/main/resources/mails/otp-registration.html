KFG OTP 등록

1. OTP 등록을 마치면 메일을 삭제하기 바랍니다.
2. 비책월드 초기화 화면에서 전송받은 이메일주소, 전화번호, TOTP 코드(6자리 예제 : 123456)를 입력해 주세요
3. TOTP를 입력 안하시면 다시 처음부터 진행됩니다.
4. 초기화 후 비번은 주민번호상 생년월일 6자리 + kfga 입니다.(예 : 801231kfga)
5. 로그인 후 반드시 비번을 변경해 주시기 바랍니다.
6. TOTP는 시간 안에 입력해 주셔야 합니다.

Google 인증기 다운로드 및 설치

1. 각 사용기기에 맞는 앱을 설치하시기 바랍니다.
2. 설정 시작하기를 누르거나 바로 바코드 스캔을 누릅니다.
3. QR코드를 인식하면 인증기가 등록됩니다.
4. 비책월드 비번 초기화 화면에서 전송받은 이메일주소, 전화번호, TOTP 코드(6자리 예제 : 123456)를 입력해 주세요
5. TOTP는 시간 안에 입력해 주셔야 합니다.
